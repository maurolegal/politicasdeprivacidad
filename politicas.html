Política de Privacidad - Asistente de IA para WhatsApp Web

Fecha de vigencia: 12 de mayo de 2025

Esta Política de Privacidad describe cómo la extensión "Asistente de IA para WhatsApp Web" (en adelante "la Extensión") maneja la información de los usuarios. Al instalar y utilizar esta extensión, aceptas los términos descritos a continuación.

1. Información que recopilamos

  • Mensajes de WhatsApp Web: La extensión accede a los mensajes visibles en la interfaz de WhatsApp Web con el único fin de analizarlos y generar respuestas automáticas mediante inteligencia artificial. Esta información no se almacena, ni se transfiere a servidores externos fuera de los servicios especificados por el usuario (como OpenAI).
  • Clave API del modelo de IA: El usuario proporciona una clave API (por ejemplo, de OpenAI), la cual se almacena localmente en su navegador mediante chrome.storage.local y se utiliza únicamente para procesar solicitudes de generación de respuestas. No se comparte con terceros ni se transfiere fuera del entorno del usuario.

2. Uso de la información

La información recopilada se utiliza exclusivamente para:

  • Generar respuestas automáticas dentro de WhatsApp Web.
  • Ejecutar correctamente la funcionalidad de la extensión.

3. No recopilamos ni usamos:

  • Datos financieros.
  • Información sanitaria.
  • Ubicación GPS.
  • Historial web o actividad fuera de WhatsApp Web.

4. Compartición de información

No vendemos, alquilamos ni compartimos información personal o de mensajes con terceros.

5. Seguridad

La extensión utiliza los mecanismos de almacenamiento local provistos por el navegador y no mantiene servidores propios. Los datos permanecen bajo el control del usuario.

6. Derechos del usuario

Los usuarios pueden eliminar su configuración en cualquier momento desde el panel de la extensión. Además, pueden desinstalar la extensión para detener toda recopilación de información.

7. Cambios a esta política

Nos reservamos el derecho de actualizar esta política de privacidad. Se notificarán cambios significativos a través de la página de la extensión en Chrome Web Store.

8. Contacto

Para dudas o inquietudes sobre esta política, puedes escribir a: tu-correo@ejemplo.com

Esta política cumple con las Políticas del Programa para Desarrolladores de Chrome Web Store.
